# Simple Project CLI

A simple CLI application for generating frontend typescript project starters on the windows platform.

## Install
  ```bash
  yarn add --global simple-project-cli
  ```

### Use

```bash
project-cli [project name] ([--template-flag] --specific-flag) 
```

#### Templates Flags:

  - Basic
    - `--basic`
    - `-b`
  - Canvas
    - `--basic`
    - `-b`
  - React
    - `--react`
    - `-r`

##### Specific Flags:

  - React Router 
    - `--router`
    - `-rr`
  - React Hook Forms
    - `--forms`
    - `-rf`
